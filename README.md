# Getting Started with Create React App

Working Demo Dev - [PathFynder](https://pathfynder-ca.web.app/).

## Run Project Locally

1. ### `npm install`
2. ### Create .env and Paste ENV Variables in .env file
3. ### `npm run dev`


## Run Tests

### `npm test`

## Production Build

### `npm run build`

Builds the app for production to the `dist` folder.\

## Run Lint

### `npm run lint`
